# Doomscroll Detector💀

**Sebuah Program yang akan mendeteksi kefokusan seseorang yang bisa dilihat dalam sebuah kamera**

---

## 📸 View (Split Screen)
Program ini pake **Split Screen Mode**:
- **Kiri:** Muka mu (Monitoring).
- **Kanan:** Layar Hitam (Standby) -> Berubah jadi **Tengkorak Joget** kalau kamu gak fokus!

---

## 💻 Syaratnya
- Python 3.9 - 3.12
- Webcam
- Speaker (biar kedengeran suaranya)
